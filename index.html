Website Performance Analyzer

A web-based tool that allows users to analyze the performance of any website. The analyzer retrieves key performance metrics—such as Performance Score, First Contentful Paint, Speed Index, Largest Contentful Paint, Time to Interactive, Total Blocking Time, Cumulative Layout Shift, and Time to First Byte—using the Google PageSpeed Insights API and Navigation Timing API. It also displays a live preview of the webpage layout and keeps a history of recent analyses.

Technologies Used: HTML, CSS, JavaScript, Google PageSpeed Insights API, Navigation Timing API
